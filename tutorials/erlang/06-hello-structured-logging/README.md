# 06 - Hello Structured Logging

Structured JSON logging in Erlang using the OTP `logger` framework and a custom formatter.

## What it covers

- Writing a custom `logger` formatter callback (`json_formatter`)
- Emitting structured JSON log lines with `jsx`
- Attaching metadata to log messages
- Configuring `logger` via `sys.config`

## Run

```bash
chmod +x run.sh
./run.sh
```

# 06 — Hello Structured Logging

JSON log objects via Elm ports.

## What you learn

- Defining Elm ports for outbound commands
- Encoding structured JSON with `Json.Encode`
- Subscribing to ports in JavaScript
- Testing JSON encoders by round-tripping through decode

## Run

```bash
bash run.sh
```

Open `index.html` in a browser and check the developer console. Each button click emits a JSON log line.

# 01 — Hello World (C3)

Basic greeting function with optional name argument.

## Prerequisites

C3 is an experimental language. Install `c3c` from <https://c3-lang.org>.

## Usage

```bash
make build   # compile
make test    # run tests
make e2e     # run end-to-end
```

If `c3c` is not installed, commands degrade gracefully with an informational message.
